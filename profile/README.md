# SPZ Ecosystem Organization Profile

Welcome to the **SPZ Ecosystem** GitHub organization! This repository contains organization-wide configuration, community health files, and documentation for the SPZ (3D Gaussian Splatting Compression) open-source ecosystem.

## About SPZ Ecosystem

SPZ Ecosystem is dedicated to developing and maintaining open-source tools for the **Khronos SPZ 3D Gaussian Splatting compression specification**. Our goal is to make high-quality 3D content compression accessible, efficient, and compatible across the entire 3D graphics and AIGC industry.

Key projects:
- `spz-entropy`: A high-performance entropy coding library optimized for 3D Gaussian Splatting data, fully compliant with the Khronos SPZ specification.
- `spz-to-gltf`: A toolchain for converting 3D Gaussian Splatting scenes into standard glTF/GLB files with SPZ compression.
- Community-driven extensions and integrations for popular 3D engines and frameworks.

We welcome contributions from everyone! Whether you're a developer, researcher, or industry partner, there are many ways to get involved:

2.  **Issue Reports**: Help us improve by reporting bugs or requesting features.
3.  **Documentation**: Improve our guides, tutorials, and API references.
4.  **Community Support**: Answer questions in discussions and help new contributors.

For detailed instructions, please read our [Contributing Guide](./CONTRIBUTING.md).
